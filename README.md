💱 Currency Converter (Frontend with Live API)

📌 Description
This is a Currency Converter web application built using HTML, CSS, and JavaScript.
It uses a real-time exchange rate API to convert currencies instantly with a clean and user-friendly interface.

🛠️ Technologies Used
HTML5
CSS3 (Flexbox layout)
JavaScript 
Fetch API (for HTTP requests)
Exchange Rate API

🎨 UI Design
Clean and minimal layout
Centered card-style container
Responsive flexbox design
Smooth user experience
Highlighted result section

✨ Features
💰 Enter amount to convert
🌍 Select currencies dynamically
🔄 Convert between multiple currencies
⚡ Real-time exchange rates
📊 Instant result display
🎯 Default: USD → INR
